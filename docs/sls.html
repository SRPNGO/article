三柳市官方网站(建设中)

简介

三柳市是星球阁mc城市群的第二大城市，它由Acubens - A于2023年1月底建立。

行政区划

三柳市目前下辖七个县区，分别是：

  • 柳城区，位于出生点周边1km内区域，是最早开建的地方。柳城区下辖核心区主要以公司大厦为主。
  • 柳阳区，位于x = 10000，z = 0处，此区专门容纳外人建设街镇，目前下辖三个街道。
  • 燕荷区，位于x = 1400～3000处，主要由银杏和qn建设，建筑以小区为主。
  • 卢云县，由室宿一在x = 0,z = -3000处建设，该县下辖北宁农业开发区位于z = -6800处，以农业发展为主。
  • 新柳区，位于x ≤ 600，z ≥ 800处，主要由室宿一和acu共同建设，此外该区下辖新南镇均从Maugfy的城市引进。
  • 柳北区，主要下辖运河商务区，以公司大楼为主。
  • 市直辖旋风街道，位于出生点正南方2000格，由天白建设，以大楼为主。

特色建筑

  1. 只因东一区。这是三柳第一座建成的小区，总容纳144户，位于只因东路地铁站，坐标（-400,0）
  2. 平安里。这是柳城区最大的小区，总容纳1368户，主要由银杏建设，位于平安里地铁站（700,-500）
  3. 地锅公园，位于（700,0），是三柳第一座森林公园。
  4. 三柳市第一中学，位于（1000,0），初中部由Acubens - A建设，高中部由qn建设。
  5. 三柳东站。这是正在建的三柳市最大的高铁站，站点包含高铁场与城际场，其中高铁未来将通至红石城银燕北站与红石北站。
  6. 核心区大厦群，主要由白菜，qn，lan等人建设，这里高楼耸立，但依然有较多区域没有开发，未来会进一步开发。
  7. 大剧院，位于（300,-500）附近。曾经2023 - 2024年春节表演在此举行。
